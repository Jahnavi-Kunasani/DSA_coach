# DSA Coach AI

An AI-powered DSA mentoring platform that analyzes a user's problem-solving approach and provides personalized feedback.

## Features

- AI evaluation of DSA solutions
- Complexity analysis
- Personalized hints
- History of previous submissions
- User authentication
- Profile page

## Tech Stack

- React.js
- Express.js
- Node.js
- JavaScript
- REST APIs

## Installation

```bash
npm install
npm run dev
