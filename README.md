# DSA Coach AI

An AI-powered DSA mentoring platform that evaluates users' problem-solving approaches and solutions, providing personalized algorithmic feedback, complexity analysis, and improvement suggestions.

---

## Features

- AI-powered evaluation of DSA solutions
- Personalized feedback on problem-solving approach
- Time and Space Complexity Analysis
- Code quality assessment
- Responsive React-based UI

---

## Tech Stack

### Frontend
- React.js
- Vite
- CSS

### Backend
- Node.js
- Express.js

- REST API

## Installation

### Clone the repository

```bash
git clone https://github.com/Jahnavi-Kunasani/DSA_coach.git
```

### Move into the project

```bash
cd dsa-coach-ai
```

### Install dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file in the project root.

Example:

```env
GEMINI_API_KEY=your_api_key_here
```

### Run the application

```bash
npm run dev
```
